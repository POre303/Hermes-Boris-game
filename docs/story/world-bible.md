# Hermes-Boris-Game World Bible (v0.1)

> 用途：剧本写作 + 资产制作 + 章节对齐的"真理之源"
> 对标：《黑森町绮谭》（拾英工作室，2 人，2019）
> 状态：v0.1 草案，可迭代

---

## 1. 核心定位

**一句话**：现代日本普通人被"异常事件"吸入一个飘忽于 1980s 平成泡沫经济余韵的虚构小镇（黑森町），在三段怪谈（鹿骨 / 列车 / 映画）中揭开"被遗留的平成记忆"的真相，最终回到现实。

**调性关键词**：神秘 / 压抑 / 余韵 / 失去 / 陪伴 / 真相

**绝不**：
- ❌ 战斗 / 技能 / 装备
- ❌ 开放世界 / 沙盒
- ❌ 多结局分支（只做普通 + 真结局 2 版）
- ❌ 多周目数值成长

---

## 2. 时代与时空

```
表世界：2010s 现代日本（地名真实，参考 JR 中央线沿线 / 奥多摩 / 高尾山一带）
里世界：1980s 平成泡沫经济余韵（地名虚构 = "黑森町"）
入口：表世界某天主角经历"异常事件"（如错过末班车/手机信号消失/走过长隧道）
       → 进入"黑森町"（不会立刻意识到是异常）
出口：每章结尾 + 尾声，主角回到表世界，但携带"平成记忆"
```

**时间感**：
- 表世界是线性的（一次会话内）
- 里世界是**回环的**（同一黄昏反复出现，主角靠对话察觉）
- 泡泡经济元素：旧报纸 / 蒸汽 / 鹿雕 / 旧式电话亭 / 黑白电视 / 老车站 / 旧胶片

## 4. 表层 vs 里层（最重要）

```
表层（玩家直觉）：
  现代主角闯入神秘小镇 → 遇怪谈 → 解谜 → 脱出

里层（剧本真相）：
  1980s 泡沫经济时代某小镇（"黑森町"）的年轻人，
  因时代变迁被"遗留"在异常空间。
  主角进入的不是"怪谈"，而是他们的"记忆场"。
  真相 = 主角在帮助这些记忆"完成未竟之事"。
```

**写作铁律**：
- **表层故事完整可独立读懂**（不解里层也不影响通关）
- **里层只在道具 / 对话细节里暗示**（不直接说破）
- **真结局揭示里层全部真相**（普通结局只给表层收束）

## 5. 主角：`nori`（占位名，可调）

**基础设定**（刻意留白，利于代入）：
- 性别 / 年龄 / 职业 / 外貌：留白
- 玩家能自由想象"我"
- 名字 `nori` 仅出现在存档 / 系统 UI，不在对话里自报

**动机**：被迫进入异常空间，想找到出口
**弱点**：体力极弱（无战斗），但观察力敏锐
**口头禅**：基本没有——用沉默 + 旁白

**对话风格**：
- 短句（5-15 字/句），不抢话
- 关键决策点用 2-3 个选项呈现，玩家不输入自由文本
- 旁白为第一人称"我"，讲感受不讲分析

---

## 6. 引导者：`rin`（占位名，可调）

**基础设定**：
- 外表：年龄模糊（看起来 15-20 岁），穿 80s 学生制服
- 自称"比你先到"——其实是小镇里最早被遗留的记忆之一
- 真实身份（真结局揭示）：1991 年失踪的少女，"黑森町"集体记忆的人格化

**功能**：
- Ch.1：给方向 + 给谜语（"灯笼的颜色会告诉你"）
- Ch.2：陪伴但不主导（"你选哪条路"）
- Ch.3：真相揭示后消失（"我去我该去的地方了"）

**对话风格**：
- 谜语式（"答案在路上，不在我嘴里"）
- 不解释自己（玩家问也不正面答）
- 偶尔露出不属于这个时代的词（"巨乳""バブリー"——Ch.3 才被理解）

## 8. 全章节大纲（4-5h）

### 序章：黑森町的入口（15-20 min，无谜题）

```
场景：现代城市（黄昏） → 主角错过末班车 → 走过长隧道 → 进入黑森町
关键事件：
  - 偶遇 rin（"你也来了"）
  - 看到旧报纸（"黑森町：泡沫经济时代繁华小镇，1991 年集体失踪"）
  - 村长出现（"外乡人……你不该在这时候来的"）
调色板：tokyo_heisei
关键道具：旧报纸
```

### Ch.1 鹿骨怪谈（60-75 min）

```
场景：村庄（RPG Maker 风） + 神社 + 鹿雕工坊 + 林间小道
三幕：
  1. 环境建立 + 谜题 1（灯笼颜色匹配 4 个）—— 找到神社入口
  2. NPC 接触 + 谜题 2（鹿骨收集 5 个）—— 解锁失踪女孩真相
  3. 高潮追逐（鹿骨妖怪）—— 主角交付鹿骨，妖怪消散
收束：真相揭示 = 失踪女孩的灵魂安息，村庄恢复一点生气
章尾：主角被传送到 Ch.2 入口
调色板：tokyo_heisei
关键道具：鹿雕、风铃、鹿骨
```

### Ch.2 猫之列车（60-75 min）

```
场景：旧式火车站 + 列车走廊 + 车厢 + 列车员室
三幕：
  1. 环境建立 + 谜题 1（车票匹配 4 个）—— 找到正确车厢
  2. NPC 接触（列车员 + 猫群）+ 谜题 2（信号灯时序）—— 列车启动
  3. 高潮（列车穿越时空隧道）—— 窗外闪过 1980s 风景
收束：列车抵达终点站，主角下车看到"1991 年"的车站
章尾：被传送到 Ch.3
调色板：train_night
关键道具：猫铃、车票、列车时刻表
```

### Ch.3 妖怪映画（75-90 min）

```
场景：剧院外观 + 大厅 + 放映室 + 舞台后
三幕：
  1. 环境建立（剧院老板的独白）+ 谜题 1（舞台道具布置）—— 解锁放映室
  2. 放映老板未完成的胶片 + 谜题 2（胶片序列排序 8 帧）—— 真相浮现
  3. 高潮（真结局分支点：选"留下修复胶片" or "回到现实"）
收束：
  普通结局：rin 消失，主角回到现代，新闻里播"某失踪 30 年的胶片重见天日"
  真结局：rin 揭示自己身份（"我就是你正在修复的那个人"），主角见证"修复"
调色板：theatre_warm
关键道具：胶片、剧本手稿、放映机
```

### 尾声：回到 2010s 的雨天（15-20 min，无谜题）

```
场景：现代城市（雨天） + 主角家
关键事件：
  - 主角醒来，发现手里握着一张旧胶片（或鹿雕/猫铃）
  - 手机显示：2026 年，新闻里播"泡沫经济时代影像修复展"
  - 制作名单（黑屏滚动 + 复古 BGM）
  - 二周目入口：标题屏加"continue from true ending?"选项
调色板：theatre_warm
```

## 9. 主题与情感节奏

**单章情感曲线**：
```
  神秘 ─→ 紧张 ─→ 共情 ─→ 真相 ─→ 释然
  头5min   第一幕    第二幕    高潮    收束
```

**全游戏情感曲线**：
```
  序：神秘
  Ch.1：神秘 → 压抑 → 释然（牺牲主题）
  Ch.2：紧张 → 陪伴 → 离别（陪伴主题）
  Ch.3：期待 → 真相 → 抉择 → 余韵（真相主题）
  尾声：平静 → 余韵 → 二周目暗示
```

**关键词权重**：
- 序章：`困局` `邂逅`
- Ch.1：`怪谈` `牺牲` `村庄`
- Ch.2：`离别` `陪伴` `时间错位`
- Ch.3：`真相` `自由` `电影`
- 尾声：`余韵` `回归`

## 10. 写作风格与禁区

**写作风格**（来自 CLAUDE.md 主题约束）：
- 对话 **30-50 字/页**，不超 3 行
- 旁白简洁（一两句，不解释）
- 谜题提示 1 句话（不超过 20 字）
- 道具 / 地点描述 1 句话（不超过 30 字）

**禁区**：
- ❌ 主角做英雄行为（推回鹿骨怪 / 拯救列车员等）
- ❌ 大段内心独白
- ❌ 反转反转（一次真相揭示就够）
- ❌ 现代梗 / 网络用语（破坏时代感）
- ❌ 直接说破里层（只能暗示）
