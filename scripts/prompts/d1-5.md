## 3. 已知陷阱（CLAUDE.md 第 95-114 行，全 17+1 条必读）

完整列表见工作目录的 `CLAUDE.md`。这里只列**对 D1-5 相关的**：

- **pnpm only**（不要 npm/yarn）
- pnpm 11 的 `onlyBuiltDependencies` 必须在 `pnpm-workspace.yaml` 的 `allowBuilds` 块，不在 `package.json#pnpm`（会被忽略 + WARN）
- pnpm 11 对 native 包（electron、esbuild、@biomejs/biome）postinstall 要 `allowBuilds` 批准，不批直接 exit 1
- electron 二进制国内 `ECONNRESET` → `.npmrc` 加 `electron_mirror=https://npmmirror.com/mirrors/electron/`
- **相对路径 off-by-one**：`src/renderer/src/palette.ts` 到 `assets/` 是 `../../../../assets/...`（**4 层不是 3 层**），容易抄错
- web tsconfig 不含 Node 类型，要导出共享 API 用字面量联合（`'win32' | 'darwin'`），不直接用 `NodeJS.Platform`
- `import x from './foo.json'` 在 web tsconfig 下需要 `.d.ts` 声明 `declare module '*.json'`，否则 `Cannot find module`
- `Proxy({} as T, handler)` TS2740 缺一堆属性，必须 `as unknown as T` 中转
- biome `lint/style/noCommaOperator` 禁 `(a, b)` 形式；侧效一句话能展开就展开
- biome `lint/complexity/useLiteralKeys` 禁 `process.env['FOO']`；直接 `process.env.FOO`
- `.claude/**` 加到 `biome.json#files.ignore`（CRLF 不被 biome format 改，CI 必挂）
- **vitest happy-dom 15 不实现 `CanvasRenderingContext2D`**（`getContext('2d')` 返回 null）→ 测试必须注入 Proxy mock
- 测试 mock 暴露的 `__press` / `__pressed` 辅助方法，类型上必须把 `ctx.input` 扩成 `MockInput`（extends `InputSnapshot`）；直接当 `InputSnapshot` 用 TS2339
- 对话/动画测试要"按帧推进"：state 内部时钟 + `update(ctx, dtMs)` 多次调用，再 `exit()` 拿决策；不要把 `exit` 当一次性接口
