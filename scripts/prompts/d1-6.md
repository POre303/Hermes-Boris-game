## 2. 任务 D1-6：集成 fusion-pixel-font 字体

### 2.1 背景

原计划用 Zpix（中文像素字体），但 **Zpix 是专有字体**（README 写"具有完全自主知识产权"），商用 $1000/产品，**不能用于商业分发游戏**。**改用 fusion-pixel-font**（OFL-1.1 + MIT，TakWolf 维护），详细对比见 `docs/research/zpix-font.md`（293 行，4 个候选对比 + 许可证验证 + 集成方案）。

### 2.2 必做步骤（参考 zpix-font.md 末段，预计 ~23 min）

1. **下载 + 解压**（~2 min）
   - https://github.com/TakWolf/fusion-pixel-font/releases（最新 v2026.05.07）
   - 下载 `fusion-pixel-font.zip`（**12px proportional woff2** = 3626 KB）
   - 解压找到 `fusion-pixel-12px-proportional.woff2`

2. **字符子集化**（~3 min，**必须**做，否则 3626 KB 太大）
   - 工具：`fonttools` (Python) 或 `subset-font` (npm)
   - 目标字符：GB2312 一级 3755 字 + Latin + 标点
   - 命令参考 zpix-font.md "字符子集化方案" 节
   - 预期产物：~500-800 KB 的 subset woff2

3. **放文件**：
   - `assets/font/fusion-pixel-12px-proportional-subset.woff2`

4. **改 HTML / CSS**（~5 min）
   - `src/renderer/index.html` 加 `@font-face` 声明
   - 全局 CSS 改 `font-family: 'fusion-pixel', 'Press Start 2P', monospace`

5. **改 dialog.ts**（~6 min）
   - 文件：`src/ui/dialog.ts`（CLAUDE.md 文字系统约束已指定）
   - 改用 `fusion-pixel` 渲染对话
   - 注意：line-height / 字符宽可能跟 Press Start 2P 不同，需要调整

6. **CREDITS.txt**（OFL 署名要求，~2 min）
   - 文件：`assets/font/CREDITS.txt`（OFL-1.1 要求附带）
   - 内容：
     ```
     fusion-pixel-font by TakWolf
     Licensed under OFL-1.1 (https://scripts.sil.org/OFL)
     Source: https://github.com/TakWolf/fusion-pixel-font
     ```

7. **跑门禁**：
   - `pnpm typecheck && pnpm test`（全过）
   - `pnpm build`（成功）
   - Playwright 视觉回归（如果已配）—— 检查中文对话实际渲染

8. **Commit（1 个）**：
   - 标题：`feat: 集成 fusion-pixel-font 字体 (D1-6)`
   - body 写改了什么 + 跑通的 test 结果

## 3. 已知陷阱（CLAUDE.md 第 95-? 行，全 21+1 条必读）

完整列表见工作目录的 `CLAUDE.md`。**重点关注**：

- **pnpm only**（不要 npm/yarn）
- pnpm 11 的 `onlyBuiltDependencies` 必须在 `pnpm-workspace.yaml` 的 `allowBuilds` 块
- electron 二进制国内 `ECONNRESET` → `.npmrc` 加 `electron_mirror=...`（**已加**）
- **相对路径 off-by-one**：`src/renderer/src/palette.ts` 到 `assets/` 是 `../../../../assets/...`（**4 层**），`index.html`（在 `src/renderer/`）到 `assets/font/` 是 `../../assets/font/...`（**2 层**），容易抄错
- web tsconfig 不含 Node 类型
- `import x from './foo.json'` 在 web tsconfig 下需要 `.d.ts` 声明
- `Proxy({} as T, handler)` 必须 `as unknown as T`
- biome `lint/style/noCommaOperator` / `lint/complexity/useLiteralKeys` 禁某些形式
- `.claude/**` 加到 `biome.json#files.ignore`
- **vitest happy-dom 15 不实现 `CanvasRenderingContext2D`** → Proxy mock
- 测试 mock 暴露的 `__press` / `__pressed`，`ctx.input` 要扩成 `MockInput`
- 对话/动画测试要"按帧推进"
- **electron-builder 24 Windows symlink**：开发模式可以，但生产 pack 要 Developer Mode
- PowerShell heredoc `"` 行被解析成 arg（用 %TEMP% + git commit -F 绕）
- `.git/` 路径被 safety hook 拦（走 git CLI 不直读）
