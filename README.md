# Hermes-Boris-Game

纯文字 + 16-bit 视觉像素风格的 Windows 独立游戏，对标 Tales of the Black Forest。
开发节奏：每日 5 次 commit。
工作流：Boris Cherny 的 loops 模式。

## 跑起来

```powershell
pnpm install
pnpm dev
```

## 构建

```powershell
pnpm build
```

构建产物在 `dist/`，目标 < 1GB。
